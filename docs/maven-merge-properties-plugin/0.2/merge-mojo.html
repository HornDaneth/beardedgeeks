merge-properties:merge

Full name:

org.beardedgeeks:maven-merge-properties-plugin:0.2:merge

Description:

Merges multiple properties files into one.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
merges Merge[] - The properties files to merge.
Usage: 
<merges>
  <merge>
    <targetFile>${build.outputDirectory}/application.properties</targetFile>
      <propertiesFiles>
        <propertiesFile>src/main/config/${property}/application.properties</propertiesFile>
        <propertiesFile>src/main/config/extended/application.properties</propertiesFile>
      <propertiesFile>src/main/config/default/application.properties</propertiesFile>
    </propertiesFiles>
  </merge>
</merges>
